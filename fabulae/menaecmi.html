| nī mala nī | stulta siēs |

| nī indomita im|posque animī |

| quod uirō esse odiō uideās |

| tūte tibi odiō habeās. |

| praeterhāc | sī mihī | tāle post | hunc diem |

| faxsis fax|sō forīs | uidua uī|sās patrem. |

| nam quotiēns foras īre uolō |

| mē retinēs reuocās rogitās |

| quō ego eam - quam rem agam - quid negōtī geram |

| quid petam - quid feram - quid forīs ēgerim. |

| portitōrem domum dūxsī ita omnem mihī |

| rem necesse ēloquī est quidquid ēgī atque agō. |

| nimium ego tē habuī dēlicātam - nunc adeō ut factūrus dīcam. |

| quandō ego tibi ancillās penum |

| lānam aurum uestem purpuram |

| bene praebeō nec quicquam egēs |

| malō cauēbis sī sapis |

| uirum opseruāre dēsinēs. |

| atque adeō nē mē nēquīquam seruēs ob eam industriam |

| hodiē dūcam scortum ad cēnam atque aliquō condīcam forās. |
